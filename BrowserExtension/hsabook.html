This must always be unchecked as the countdown is also checked server-sided
  1. First add a user (below here).
  2. To mark a course for booking, go to the corresponding course site and mark the course by clicking the button on the right of the table row.
  3. When clicking the ARM button on the top of a course site, the site will refresh continuously until the marked courses are bookable, and then the book buttons will be pressed automatically.
  4. If Fill form is checked, the booking form will be automatically filled with your user data.
  5. If Submit immediately is checked, the filled booking form will be automatically submitted, which is a lot faster than clicking the submit button by hand.
  6. As the javascript code slows down a lot when a tab is not visible, it is recommended to keep all course sites that you want to book in visible tabs. E.g. when arming multiple courses, open the course sites in separate windows and position the windows next to each other so they all are visible. This ensures a fast booking.
